Manual de Usuario

Introducción

Bienvenido a la guía de usuario de nuestra aplicación web de modulación de señales. Esta herramienta está diseñada para facilitar la comprensión y la visualización de diferentes técnicas de modulación (ASK, FSK, BPSK, 4PSK, QPSK, 8PSK, 4QAM, y 8QAM). Proporciona una interfaz interactiva donde puedes explorar señales digitales y moduladas con un enfoque educativo.

Características principales

  • Generación de señales digitales: Basada en una secuencia binaria introducida por el usuario.
  • Visualización de señales moduladas: Soporte para ASK, FSK, BPSK, 4PSK, QPSK, 8PSK, 4QAM y 8QAM.
  • Diagrama de constelación: Representación gráfica para modulaciones PSK y QAM.
  • Tablas de verdad: Indicando las fases asociadas a cada combinación de bits.
  • Interfaz interactiva: Permite limpiar la pantalla y reiniciar las configuraciones con un solo clic.

Instrucciones de uso

  1. Selecciona el tipo de modulación deseada en el menú desplegable de Modulación.
  2. Ingresa una secuencia de bits binarios (por ejemplo, 10101010) en el campo correspondiente.
  3. Haz clic en el botón Dibujar señal para generar las gráficas correspondientes.
  4. Para modulaciones PSK o QAM, revisa el diagrama de constelación y la tabla de verdad.
  5. Utiliza el botón Limpiar pantalla para reiniciar el proceso y comenzar de nuevo.

Descripción de los elementos

  • Menú de navegación: Accede a diferentes secciones de la aplicación.
  • Selector de modulación: Menú desplegable para elegir el tipo de modulación.
  • Campo de texto: Introduce la secuencia binaria (0s y 1s).
  • Gráfica de señal digital: Visualización de la señal ingresada.
  • Gráfica de señal modulada: Representación de la señal según el tipo de modulación.
  • Diagrama de constelación: Muestra las fases asociadas a combinaciones de bits (solo PSK y QAM).

Notas importantes

  • Asegúrate de ingresar solo bits binarios en el campo de texto.
  • Si ingresas caracteres inválidos, la aplicación te mostrará un mensaje de error.
  • Para modulaciones PSK y QAM, utiliza secuencias que coincidan con los esquemas soportados (2 bits para 4PSK, 3 bits para 8PSK, etc.).
  • Los gráficos se actualizan automáticamente cada vez que se realiza un nuevo cálculo.
